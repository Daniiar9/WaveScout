# Feedback Loop

The feedback loop is a placeholder for future learning from creator outcomes.

Future loop:

creator contacted -> creator replies -> content posted -> comments/signups tracked -> angle score updated -> creator score updated -> next wave recommendations improve.

## Manual Feedback Fields

- creator handle
- content angle
- outreach status
- creator response
- post URL
- views
- likes
- comments
- clicks
- signups
- notes

## Current Safety

No external analytics integrations exist yet. Feedback is manually ingested and summarized offline.
