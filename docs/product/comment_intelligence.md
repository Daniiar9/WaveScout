# Comment Intelligence

Comments matter because they show whether an audience is only reacting to AI hype or asking implementation-level questions that indicate real buyer or user intent.

## Intent Types

- asks_for_tool
- asks_for_link
- asks_for_tutorial
- shares_use_case
- expresses_pain
- expresses_skepticism
- tags_friend
- generic_hype
- creator_to_creator
- irrelevant

## Buyer And User Intent Examples

- "Can this connect to Notion?"
- "Does it work with Slack?"
- "I need this for my CRM."
- "What tool is this?"
- "Can you make a tutorial?"
- "How would this work for a small agency?"

## Low-Quality Hype Examples

- "first"
- "AI is crazy"
- "bro what"
- "follow me"
- "chatgpt changed my life"

## How Comments Affect Scoring

Comment intent quality contributes up to 20 points. High-quality comments can lift a small creator above a larger account. Mostly generic or irrelevant comments reduce confidence and can trigger a reject decision.

## Analysis Scope

Comment analysis is designed for shortlisted or manually imported creators. It is not intended to crawl random creators or scrape TikTok at scale.
